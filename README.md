# InviFlow - Investment Portfolio Platform

A Django-based investment portfolio platform with Google Sheets integration and automated invoice generation.

## Features

- ✅ User portfolio management with stock holdings
- ✅ Deposit processing with automatic PDF invoice generation
- ✅ Google Sheets API integration for data synchronization
- ✅ Cron-ready automation scripts
- ✅ Clean, maintainable code structure
- ✅ Docker support for easy deployment

## Tech Stack

- **Backend**: Django 4.2, Python 3.11
- **Database**: PostgreSQL
- **Cache/Queue**: Redis, Celery
- **External APIs**: Google Sheets API
- **PDF Generation**: ReportLab
- **Container**: Docker, Docker Compose
